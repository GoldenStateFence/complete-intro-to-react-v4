# Welcome

Welcome to the code behind Introduction to Web Development v2. The course can be found [here][course]. It best explains itself so read about it there.

## Devlopment

To develop locally

1. git clone this repo
1. `npm install`
1. `npm run develop`

To publish

1. `npm run publish`
